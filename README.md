🍽️ Hotel Senapati – Restaurant Website
A fully responsive restaurant website built using HTML5 and CSS3.
This project represents a modern hotel/restaurant landing page with menu display, chef section, about section, and contact form.

📌 Project Overview
Hotel Senapati is a static front-end website designed for a restaurant.
It includes:
-Hero section with background image
-Navigation bar (Desktop + Mobile responsive)
-Menu section (Breakfast, Lunch, Dinner)
-Chef section
-About Us section
-Contact section with form
-Footer with smooth scroll
The website is fully responsive using CSS media queries.

🛠️ Technologies Used
-HTML5
-CSS3
-Google Fonts (Poppins)
-Font Awesome Icons
